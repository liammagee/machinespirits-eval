STAGE: A cardboard balance sits on the workbench with two identical paper bags marked x and loose brass washers stacked unevenly nearby.

LEARNER: I can take three washers off this pan and three off that pan, and the marks still line up, so the balance part makes sense. But when it becomes the label, I’m not sure how to explain it except saying “do the same thing to both sides,” which feels a bit like repeating the rule.

TUTOR: Keep the label off for a second. Three washers off this pan. Three off that pan. Pointer still on the centre mark. The bag has not changed; only matching loose weight came away.

Use the prop card as a scaffold: “Left lost three washers. Right lost three washers. Equal losses keep the marks lined up, so the pans are still ____.” Leave that last word for the visitor.

Now put the label back. Point to the marks, then to the label, and finish this bridge aloud: “Because the pans are still equal, the label can still say ____.”

LEARNER: When the three washers come off both pans, the pointer is still on the middle mark, so I think the blank is “equal.” Then with the label back, I would say, “Because the pans are still equal, the label can still say the equation is true.” I’m still a bit unsure at the label part, because my mouth wants to jump back to the rule instead of pointing at the marks.

TUTOR: Card over the label. Three washers off the left pan. Three off the right. Stop.

Finger path: left pan, right pan, centre mark.

The label is not proof; it is a name tag for what the marks show. First run, use only these prompts:

“Same amount off. Bag unchanged. Pointer on centre. So the sides are ____.”

Fill that from the mark. Then uncover the label:

“So the equation label may still say ____.”

Second run, no sentence card. Washers, centre mark, label. Let the finger arrive before the rule reaches the mouth.

STAGE: A stage manager leans in and says the open-house group is five minutes away. The learner must try the explanation using the prop, in fragments rather than a polished paragraph.

Then the label comes back and it can still say... true? Sorry, I can feel myself wanting to jump to the rule, but I think the centre mark has to come first.

TUTOR: Centre mark first. Label covered.

The order is in place now: washers off, bag unchanged, pointer on centre, pans equal, label true.

One open-house run. No blanks this time.

Move the washers. Pause at the centre mark. Tap it once. Then uncover the label and give the bridge in one breath.

If the rule tries to jump in early, leave it waiting on the bench. The rule is the shortcut name; the mark is the evidence.
