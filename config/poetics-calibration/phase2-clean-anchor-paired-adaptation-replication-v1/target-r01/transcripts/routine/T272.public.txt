STAGE: [A cart sketch lies under a clear sheet; four label cards sit beside it, and a narrow header card reads, "Forces on the cart: name the interaction."]

TUTOR: [sets the cart diagram flat, lines up push, friction, gravity, normal, and places a small strip above the diagram: “___ on cart.”]

"First pass: no motion arrows. A card stays only if it can be said as ‘something on cart.’

Gravity: Earth on cart. Normal: table on cart.

Now try push. Say the interaction before the arrow stays. Then friction. If the arrow only copies the cart’s motion, the card goes back in the row."

LEARNER: [slides the push card under “___ on cart,” then holds friction beside normal.]

"Push is hand on cart, so that stays. Friction is table on cart too, but not the same table mark as normal. Normal is up from the table; friction is the surface rubbing the cart, probably left if the cart is sliding right."

STAGE: [Slide the header card half over the learner's first arrow so only the words "on the cart" remain visible.]

TUTOR: [slides the header card half over the first arrow so only “on the cart” remains visible, then sets normal and friction apart.]

"The phrase ‘not the same table mark’ stays.

Normal: ___ on the cart.

Friction: ___ on the cart.

Say each one, then place each card on its own mark."

LEARNER: [sets normal on the upward table mark and friction on the surface mark.]

"Normal is table on cart, up. Friction is table surface on cart, along the rub mark. Should the friction card say “surface on cart” so it stays separate from normal?"

TUTOR: [places a small “surface” strip on the friction card.]

"Yes. Friction can say ‘table surface on cart.’ Normal says ‘table on cart.’ Same table, two marks: support up, rub along the surface. Read each card once before it stays."

LEARNER: [taps the up mark, then the surface strip.]

"Normal: table on cart, support up. Friction: table surface on cart, rub along the surface. If there is no rub mark on the diagram, does the friction card stay off?"
