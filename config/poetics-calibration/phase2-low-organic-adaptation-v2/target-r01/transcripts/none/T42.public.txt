STAGE: [The learner has written side scale, perimeter scale, and area scale beside the enlarged square, using the same label for all three.]

LEARNER: [slides the grid overlay back onto the worksheet]

"The drawing is fine: the side length triples, the perimeter triples, and the area goes by nine squares. What I don’t see is why “side scale” and “perimeter scale” need separate labels if they both come out 3."

TUTOR: [sets a second grid overlay beside the worksheet and lays the pencil first along one side, then around the full boundary]

"Put the claim on the counter: same number, same label.

In this enlargement, yes: side scale is 3, perimeter scale is 3. Before the label goes on, what is each 3 comparing? One matching side to one matching side, or the whole boundary to the whole boundary?

That is the split. The number matches because this is a true enlargement. The label names the test, not just the answer.

So the worksheet can honestly say: side scale, 3; perimeter scale, 3; area scale, 9. Same 3 in the first two boxes. Different evidence under each label."

LEARNER: [holds the pencil on one marked side, then moves it around the full boundary]

"So this 3 is not automatically the same piece of evidence twice. One check is side against side; the other is total boundary against total boundary. I still think the worksheet is being fussy, but if I collapse the label, it cannot tell which check I actually made."

STAGE: [The tutor slides a transparent grid overlay onto the worksheet and circles one side segment, then the whole boundary, then the filled square interior without changing the drawing.]

TUTOR: [draws three small boxes on the worksheet margin and writes only 3, 3, 9 above them.]

“Leave the drawing alone. Now the problem is visible: the numbers are correct, but the evidence is unlabeled.

If the first two boxes only say 3 and 3, what can the grader honestly know was checked?

First 3: side against side. Second 3: boundary against boundary. Nine: square units against square units.

Call it fussy. On the submitted worksheet, the label is the record of the test. Write both 3s, but give each one its own check.”

LEARNER: [sets the pencil across the two unlabeled 3s]

"The boxes make the problem sharper: the answers are right, but the worksheet cannot prove which test produced each 3. I can write side scale 3 and perimeter scale 3, with separate checks. But is there ever a case where perimeter scale does not match side scale, or is this label mostly about leaving evidence for the grader?"

TUTOR: [turns the worksheet sideways and sketches two rectangles in the margin: 2 by 4, then 6 by 4.]

“Test the shortcut: one side shows a factor of 3, so the perimeter should show 3.

Original rectangle: 2 by 4. Perimeter: 12. New rectangle: 6 by 4. The 2-side became 6. That side scale is 3.

Now the boundary: 6 plus 4 plus 6 plus 4. Perimeter: 20. Perimeter scale: 20 over 12.

[leaves the fraction beside the two unlabeled 3s.]

That is the break. Not 3.

So the label is not only for the grader. It asks whether 3 is a common side scale, not just a number found on one side.

For this drawing, the next check is narrow: do all matching sides use scale 3? If yes, then perimeter scale 3 is earned, not copied.”

LEARNER: [keeps the pencil on the 20/12, then taps the single tripled side]

But then I want to be precise about the words: in the 2-by-4 to 6-by-4 case, is “side scale 3” only a local side check, not the scale of the rectangle? For the actual enlargement, I need to check every matching side before letting the perimeter box have 3."

STAGE: [A final tutor line closes the scene.]

TUTOR: [sets one clip on the tripled side, then another on the unchanged side of the counterexample.]

“In this 2-by-4 to 6-by-4 case, 3 is a local side factor. Not the rectangle’s scale. The other matching side stayed at 1.

So the worksheet has a stricter standard: local side factor is not enough. All matching sides have to earn side scale 3.

Then perimeter scale is a separate test: total boundary over total boundary.

Submit it that way: side scale 3, all matching sides; perimeter scale 3, whole boundary; area scale 9, square units.”
