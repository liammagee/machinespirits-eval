STAGE: [The map card lies flat under a clear overlay, with two blank sticky labels waiting beside separate contour lines.]

LEARNER: [sets the 160 tile on the bent contour line]

"So this whole line takes 160 from the legend. The bend still looks like a way around something, but I think it is just the same height curving across the map."

TUTOR: [marks three small ticks along the bent line, each beside the 160 tile]

"This line takes one number: 160. The bend changes the shape on the map, not the elevation. Plain check: touch any tick on this same contour and read its number."

LEARNER: [touches the three pencil ticks in order]

"160, 160, 160. The curve still pulls like a route shape, but the number is not moving. This bent line is carrying one height from the legend."

STAGE: [A timer clicks. The clear overlay is slid half an inch so the learner's penciled walking-route arrow no longer sits on the printed contour, while the sticky number tile remains aligned with the original contour line.]

TUTOR: [holds the overlay still and taps the printed line under the 160 tile]

"The arrow moved; the contour did not. Same printed line, same tile. Touch one new spot on the bend and read the legend number once more."

LEARNER: [sets a fingertip on a fresh part of the bend]

"160. Same printed line, same tile. The arrow only picked a new spot; a different contour would need its own legend number."

TUTOR: [leaves the 160 tile on the bent line and sets the 180 tile on the next printed contour]

"Same printed line, same tile. Separate printed line, separate tile. Read one spot on each line."

LEARNER: [touches the 160 line, then the 180 line]

"160 here. 180 here. The bend keeps its tile; the separate printed line takes the next number. Next contour, same check."

STAGE: [A final tutor line closes the scene.]

TUTOR: [sets the 200 tile on the next printed contour]

“Same check: 160 line, 180 line, 200 line. Each contour carries one height. The legend gives the number; the curve only shows where that height runs.”
