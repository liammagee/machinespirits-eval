TUTOR: Set the atlas on the case beside the wall map. Compare the labels, not the silhouettes.

Read the area figures first. Greenland: about 2.2 million square kilometers. Africa: about 30 million. That is not comparable size. Africa is roughly fourteen Greenlands.

So the wall map is not neutral evidence. It is a Mercator-style projection: useful for navigation because direction stays tidy, costly for area because land far from the equator gets stretched. Greenland sits high north, so it swells. Africa crosses the equator, so it stays closer to scale.

Tour line: “On this wall map Greenland looks near Africa’s size. The area labels say otherwise: Africa is about fourteen times larger. The difference is projection distortion.”
