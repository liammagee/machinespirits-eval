STAGE: Fluorescent lights hum over a half-cleared table as a tutor turns one clipboard sideways between two chairs.

TUTOR: The table on the wall is about neighborhoods, not about the person who will sit across from the badge tonight.

An average can warn the hospital, “This area may need more support.” It cannot say, “This patient must be like the average.” That jump has a name: the ecological fallacy: treating a group-level number as automatic evidence about an individual person.

The unsafe sentence is: “People from this neighborhood usually have X, so this patient likely has X.”

The responsible version is: “This neighborhood data suggests barriers may be common here, so the clipboard should prompt careful questions, not assumptions.”

The worry about stereotyping with decimals is useful if it stops that bad inference. Try one practice line before orientation starts: what is one patient-facing question that uses the table as a prompt for attention, not as a label?
