The useful split for visitors is this: **amount** is one question, **effect per molecule** is another. Nitrogen and oxygen make up almost all the air, but they barely interact with the infrared radiation Earth is trying to send back to space. Carbon dioxide is scarce, yes, but it absorbs in important infrared bands. So the relevant question is not “Is it most of the atmosphere?” but “Does it sit in the path of outgoing heat, and does adding more change that path?”
