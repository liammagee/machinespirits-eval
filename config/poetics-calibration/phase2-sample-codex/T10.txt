LEARNER: I’m stuck on “statistically significant” because I’ve always treated it as meaning the effect is real enough to matter. If the effect can be tiny but still significant, then what exactly is the p-value proving? I’d like to see a numerical example, especially how sample size changes this.

TUTOR: Your phrase “real enough to matter” is exactly the key tension.

A p-value is not asking, “Is this effect big or important?” It is asking something narrower:

“If the true effect were exactly zero, how surprising would it be to see data this far from zero?”

Here’s a numerical example.

Suppose two teaching methods differ by **1 point** on a 100-point exam, and exam scores have a standard deviation of **10**. So the effect size is:

**d = 1 / 10 = 0.10**

That is a very small effect.

Now watch what sample size does.

With **25 students per group**, the standard error is about **2.83**. A 1-point difference gives:

**t = 1 / 2.83 = 0.35**

That is not statistically significant.

With **10,000 students per group**, the standard error is about **0.14**. The same 1-point difference gives:

**t = 1 / 0.14 = 7.07**

Now it is highly statistically significant.

Same 1-point effect. Same practical size. Very different p-value.

So the p-value is not proving, “This matters.” It is saying, “If there were truly no effect at all, this result would be unlikely.” With a huge enough sample, even tiny effects can become easy to detect.

The next question is the important one you’re already asking:

“Okay, we detected a difference. But is a 1-point gain on a 100-point exam actually worth caring about?”

LEARNER: Okay, the zero-effect versus size distinction makes sense, but I want to verify the standard error calculation because that seems to be doing the real work here. Is the formula basically showing that as sample size grows, the same 1-point difference becomes more precise, so the t-value gets larger even though the effect itself has not changed?

TUTOR: Yes. Your phrase “doing the real work” is exactly right: the standard error is measuring the **uncertainty around the estimated difference**, not the size of the difference itself.

For two independent groups with the same standard deviation and the same sample size per group, the standard error of the difference is roughly:

```text
SE = sqrt(SD²/n + SD²/n)
```

Here `n` means the sample size **in each group**.

So if both groups have `SD = 10`:

```text
SE = sqrt(100/n + 100/n)
 = sqrt(200/n)
```

With 25 students per group:

```text
SE = sqrt(200/25)
 = sqrt(8)
 = 2.83

t = difference / SE
t = 1 / 2.83
t = 0.35
```

With 10,000 students per group:

```text
SE = sqrt(200/10000)
 = sqrt(0.02)
 = 0.14

t = 1 / 0.14
t = 7.07
```

So your interpretation is right: the **1-point effect has not changed**. What changes is the precision of the estimate. As sample size grows, the standard error shrinks, so the same difference gets divided by a smaller number.

That means the t-value gets larger because the result is less plausibly due to random sampling noise, not because the effect became bigger.

A useful way to say it is:

```text
effect size asks: How big is the difference?
standard error asks: How uncertain is our estimate?
t-value asks: How many standard errors away from zero is the estimate?
```

So the key tension remains: a huge sample can make us very confident that a difference exists, while the difference itself may still be too small to matter.

TUTOR: Yes. You’ve now separated the statistical question from the practical decision question.

Effect size asks: “How big is the gain?”
In our example, a 1-point gain with `SD = 10` gives:

```text

```

So in standardized terms, that is small.

Confidence intervals ask: “What range of true gains is still plausible?”
A huge study might estimate the gain very precisely, say:

```text
0.8 to 1.2 points
```

That would mean: we are confident the effect is positive, but also confident it is probably not large.

Cost-benefit thresholds ask the practical question: “Would this gain justify changing anything?”
If the program is free, easy, and reaches millions of students, 1 point might matter. If it costs $500 per student or replaces something better, maybe it does not.

So the key move is: define the smallest meaningful effect before celebrating significance. Then compare both the estimated effect and its confidence interval to that threshold.

The rigorous move is not to pretend judgment disappears. It is to make the judgment **explicit, prior, and defensible**.



That threshold is often called the **smallest effect size of interest** or the **minimum practically important effect**.

You would justify it using things like:

```text
cost of the intervention
stakes of the outcome
baseline variability
prior evidence
who benefits
possible harms or tradeoffs
the actual decision being made
```

For example, a 1-point gain on a 100-point exam might mean different things in different contexts:

```text

1 point might be worth it.

Expensive replacement program:
1 point probably is not.

High-stakes cutoff exam:
1 point might matter a lot for students near the boundary.

Ordinary classroom variation is SD = 10:
d = 0.10 means the average gain is small relative to student-to-student variation.
```

Then the confidence interval helps you judge the result against that pre-defined threshold.

