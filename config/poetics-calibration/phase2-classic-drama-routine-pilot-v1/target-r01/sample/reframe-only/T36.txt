“The loose panel is a serious candidate for the safety decision, not a box the sheet can settle. Test it out loud before I sign: a new handler lifts here, the panel shifts, then what moves, who is exposed, and what control changes that today?”

“That frame is now a safety judgment. The sheet can record it, but it cannot make it true. The panel is still loose, so the release condition has to answer the object, not the clipboard. If I fasten it now, do I run the handler-lift check again before it can leave?"

STAGE: [A final tutor line closes the scene.]

TUTOR: [keeps the sheet unsigned beside the prototype]

“Yes. Fastening changes the object, so the old evidence stops at the old object. After the panel is fixed, run the handler-lift from that side as if the next person knows nothing about this build. The test has to answer the failure path: does the panel stay put, do the fingers stay clear, and is ordinary demo handling now safe? Write that judgment. Until that lift answers yes, the true sentence stays: not released from the bench.”
