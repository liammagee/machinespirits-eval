STAGE: [The learner has drawn a larger square around a smaller one; the doubled side length is marked in blue, while the original square remains visible underneath.]

LEARNER: [slides the transparent grid over the marked square]

"This is double the side, so I still want to call it double. Wait, the grid is showing four of the old square inside it, not two."

TUTOR: [marks the doubled side in one color, then the doubled height in another]

"Hold both facts. The side is double. The area is not.

Count what the grid now permits: how many old squares across? How many up?

Two across, two up. So the area claim is not \(2\). It is \(2 \times 2 = 4\).

Before the test: doubling the side length makes four times the area."

LEARNER: [presses the grid flat against the counter]

"Fine, the side is double, but the grid is not letting me call the area double. It is two old squares across and two up, so \(2 \times 2\), four copies. I still dislike that my first word was wrong, but before the test: double side length means four times the area."

TUTOR: [draws one brace across the doubled width, then one up the doubled height, leaving the four old squares boxed inside]

"Keep the dislike; it is not in charge of the count. “Double” is true along this brace, then true again along that brace. Across is \(2\); up is \(2\). Say the area claim from the boxes: \(2 \times 2 = 4\). If the side factor were \(3\), what area factor would the same grid logic force?"

STAGE: [The tutor slides the transparent grid sheet half an inch so the original square appears repeated in each quadrant of the larger square.]
An earlier learner line returns to the table: "[slides the transparent grid over the marked square] This is double the side, so I still want to call it double." The pause holds while the learner decides whether that wording still stands, needs narrowing, or needs replacing.]

LEARNER: [keeps the grid pinned]

"“This is double the side, so I still want to call it double” still stands only for the side, not the area. For area it has to be two across and two up: \(2 \times 2 = 4\). If the side factor is \(3\), the same grid forces \(3 \times 3 = 9\). Annoying, but the boxes do not leave room for double."

TUTOR: [sets the blue marker along the width and the red marker along the height, then writes \(2 \times 2\), \(3 \times 3\), and only then \(k \times k = k^2\) beside the grid]

“Double” is restricted, not erased. It belongs to the side mark only.

Area must answer to both marks: across and up. The grid has already forced the rule: side scale factor \(k\), area scale factor \(k^2\).

Old side \(5\), new side \(10\). State the area factor without using the word “double.”

LEARNER: [slides the red marker against the height brace]

"Side factor is \(10/5=2\); area factor is \(2 \times 2=4\). My old label hid the second direction. Four times the old area."

STAGE: [A final tutor line closes the scene.]

TUTOR: [sets the blue marker across the width and the red marker up the height, then leaves them crossed at the corner]

"Keep that sentence: the old label hid the second direction. This mark says \(2\) across; this one says \(2\) up. The grid’s verdict is four copies. On the test: side factor \(k\), area factor \(k^2\)."
