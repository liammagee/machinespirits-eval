“The checklist says the bracket met the listed requirements. The load test says it held under that test condition. The sign-off line is a different claim: safe under foreseeable use.

Before the shop takes it, check the crooked case: load shifted to one front corner, cart pulled at an angle, bracket taking twist instead of straight load. Trace that load path. Where does the bracket bend first, and if it slips, whose hand or foot is in the failure line?

Then the sign-off line needs a defensible decision: release as tested, release with a use limit or label, add a stop, or hold it for redesign. Which one does the crooked-load case support?”

If the failure line is near the person’s hand or foot, I guess I’d be choosing between a stop and holding it, not just adding a label.”

“Not cleared for morning use yet. The sheet proves clean-load survival; the sign-off still has to cover foreseeable corner-pull use.

That distinction is the design review now. Set up the front-corner load and angled pull. Mark the first bend point, the slip path, and the hand-foot zone.

If the slip path enters that zone, a label is not the control. What evidence would show that a stop actually contains the twist, and what result would mean the bracket has to be held for redesign?”

TUTOR: [marks the sign-off line “HOLD - corner-pull evidence pending”]

“The apology is separate. The bracket is not released on an unknown bend limit.

Use these as review thresholds during the corner-pull run: the bracket mouth stays closed, the fasteners do not walk, the bracket does not keep a permanent set after release, and the slip path stays outside the hand-foot zone when the stop takes load. The stop can be a control; it cannot be a crash point for a bracket already opening.

Set witness marks before the run. Measure unloaded shape, loaded shape, and released shape. If the stop takes the twist and the bracket returns, that supports release with the stop. If the stop is struck hard, the bend keeps growing, or the bracket starts opening, that is hold for redesign, not a label.

What measurement will show return-to-shape rather than permanent set?”

LEARNER: "I think I’d measure the bracket mouth width and the witness-mark spacing before the run, then measure them again after the load is off. “Mouth width and witness marks are the release check: did the bracket come back, did the fasteners walk, did the slip path stay out of the hand-foot zone. Add bend angle because the corner-pull case is a twist case; it shows whether the stop is taking the load or the bracket is failing into the stop.

Before the run, write the shop tolerance on the sheet. ‘Enough’ cannot be decided after the bend looks small. Record unloaded angle, loaded angle at worst case, and released angle after the load is off.

If the loaded angle rises to the stop and stabilizes, and the mouth width, witness marks, and released angle return inside tolerance, that supports release with the stop. If the angle keeps climbing when the stop is hit, the mouth opens, or the released angle stays changed, the sign-off stays hold for redesign.

The facilities cart waits until those numbers are on the sheet.”
