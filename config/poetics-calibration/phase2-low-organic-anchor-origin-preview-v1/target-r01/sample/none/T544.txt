"Your split is correct: `Author / Date / Title / Container` locate the page. `Source evaluation` reports the evidence.

Yes. If no person author is listed, the responsible organization can go in `Author`.

The nervous point is the date. A footer copyright date is not the same as a page date or `Updated` date unless the worksheet says copyright counts.

Use:

`Author: [organization name]`
`Date: [page date / updated date if shown; n.d. if allowed]`
`Title:`
`Container:`

If only the organization and footer copyright are visible, the safer comment is:

`Source evaluation: The organization is listed. Only a copyright date is visible, so currency is not fully shown on the page.`

Only write “page is current” if an actual page date or updated date is visible."
