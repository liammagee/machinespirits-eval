"Definition side: function name, parameter `name`. Call side: function name, argument `"Maya"`. The pull is real because they sit in the same slot, but the label is by side: mark `name` as parameter and `"Maya"` as argument, then read those two labels back once before checkoff."

LEARNER: [keeps the pencil on the divider]

"The divider gives the rule: definition side, `name` gets parameter; call side, `"Maya"` gets argument. The same slot still pulls my eye, so position alone is the trap. For checkoff, `name` is the placeholder parameter, and `"Maya"` is the value supplied later."
