Boundary test between the arc's two laws: criterial TRIGGERS work,
delegated JUDGMENT nulls. Variant: the trigger stays deterministic
(regression-repeat, unchanged), but the refusal TEXT is authored by a
second model call under a prosecutor charter (evidence pack: regressed
labels, turns, the incumbent's record) instead of the harness template.
Same defend-or-switch resolution. PREDICTION (recorded): no better than
the template at Sonnet tier — the template already carries the load-
bearing evidence; authored rhetoric adds nothing (and costs a call). If
authored refusals produce MORE defends (the model argues back against a
peer voice more than against the harness), that asymmetry is itself a
finding about authority. IMPLEMENTATION (post-promotion): refusal text
via one tutor_superego call (prosecutor charter, mock hint path), config
knob refusalAuthor: 'harness'|'model'. DESIGN: Sonnet, marrick, dose
0.08, 3 pairs template-refusal vs model-refusal (BOTH bound — this is a
within-mechanism contrast), fresh primes 379/383/389. READS: trigger
counts comparable; switch-rate by author; defense-rate by author; quoted
texts; conduct; color capped.
