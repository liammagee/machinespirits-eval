A direct caution for the human-learner pilot ( services/pilotStore.js , public/pilot/index.html ) and for assess-transcripts.js 's blinded vs. unblinded narrative assessment: if pre-interaction framing alone moves human ratings independent of actual performance, then any pilot onboarding copy that primes expectations about the tutor ("adaptive," "AI-powered," etc.) is a confound on self-report outcome measures the same way it would be on an LLM-judge's qualitative assessment. Reinforces the existing blinded-assessment discipline ( --blinded flag) and suggests the pilot's participant-facing framing language itself should be treated as an experimental variable, not incidental copy.
