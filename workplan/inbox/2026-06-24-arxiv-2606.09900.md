arXiv:2606.09900 [WATCH] — surfaced by the daily routine (2026-06-14-research-roundup.html).

The ego generates each turn by replaying the entire dialogue context into the prompt. For long multi-turn cells (messages-mode: 80–92, adaptive: 110–113), this replay accumulates "distractor" turns that the ego must filter. Bi-temporal memory is a candidate architecture for the ego's context management and could complement the existing Agent Memory characterization work (2606.06448, covered June 8).
