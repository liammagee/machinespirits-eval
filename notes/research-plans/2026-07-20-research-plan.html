P1 · Judge-reliability contagion audit — "Contagion Networks: Evaluator Bias Propagation in Multi-Agent LLM Systems" (2606.20493)

Why: formalizes how one agent's biased judgment contaminates downstream agents that consume it as context, modeling bias spread across agent chains; proposes independent sampling and adversarial-critic-insertion as mitigations. Direct warning for a pipeline that assumes independent judges (analyze-judge-reliability.js's paired-rejudge methodology) sitting downstream of an Ego→Superego→Ego loop where a biased superego critique could compound across revision rounds — precisely the systematic drift §5.4 tries to detect from the other direction.
Do: two-part, both read-only. (1) Code audit: trace the judge-prompt construction path in services/evaluationRunner.js / the rubric evaluators to confirm the judge only ever sees the public suggestions output, never tutor_deliberation_* superego critique text — i.e. no direct contamination channel. (2) If clean, check for the paper's indirect signature instead: does judge-score variance across the divergent-superego family (cells 22-33 — suspicious / adversary / advocate × base/recog) shrink or trend in one direction as the superego framing gets more adversarial, which would look like compounding bias rather than independent judgment even without a literal context leak.

P3 · Second mechanism for the poetics κ-failure — DeFAb defeasible-abduction benchmark (2606.18557)

Why: DeFAb finds frontier-model performance on defeasible abduction — drawing a revisable conclusion from incomplete evidence, then retracting it when contradicting evidence arrives — degrades systematically as abductive complexity (revision depth) rises. Recognition-as-belief-revision is structurally the same operation the poetics critic has to detect for anagnorisis: the learner-character retracting an earlier premise. Last week's P2 (still open) already named CASPER's closed-arc bias as one plausible mechanism behind the phase-2 κ = 0.044 failure (DRAMATIC-RECOGNITION-PLAN.md line 18); DeFAb gives a second, orthogonal one — the critic may specifically under-score anagnorisis at higher revision depth, independent of arc-closedness.
Do: on the same already-labeled phase-2 calibration sample named in last week's P2 (config/poetics-calibration/phase2-*.yaml), additionally hand-code each recognition event for revision depth using DeFAb's abductive-complexity scheme, then correlate depth against anagnorisis and cathartic_closure scores alongside the CASPER closed/open-arc coding already planned — same sample, two coding passes, one combined correlation.
