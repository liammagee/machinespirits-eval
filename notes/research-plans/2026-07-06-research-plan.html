P1 · Audit ego/superego incorporation with "What LLM Agents Say When No One Is Watching" (2607.02507)

Why: a dual-channel debate framework — public utterances vs off-the-record (OTR) responses never shown to the other agent — finds that alignment-inducing social structure (role framing, audience, hierarchy) widens the public/OTR gap: agents perform convergence publicly while private reasoning diverges more under exactly the framings meant to produce agreement. The bilateral ego/superego architecture already has this structural split built in — superego critique is private (tutor_deliberation_*), the ego's revision is public (suggestions) — but nothing currently measures whether the public revision tracks the private critique or just performs having addressed it.
Do: extend services/dialogueTraceAnalyzer.js's analyzeSuperegoIncorporation() (already computes incorporation_rate) with a public/private divergence measure — critique points raised by the superego that the ego's revision does not actually address. Run it across the superego-configured cells (3-4, 7-8, and the divergent-superego family 22-33, which spans suspicious/adversary/advocate framings) to check whether the more adversarial framings (28-33, dialectical ego + divergent superego) produce genuinely tighter incorporation or just a wider performative gap — the mechanistic question 2607.02507 raises but doesn't have this architecture to test.
Evaluate: existing dialogue logs already in the DB (tutor_deliberation_*, incorporation_rate, dimension_convergence) for cells 3-4, 7-8, 22-33 — a new analysis script, no new generation. existing data

P2 · Diagnose the poetics κ failure with CASPER (2606.22454) + the narrative-theory survey (2602.15851)

Why: CASPER pairs 4,400 LLM-generated stories against human-written twins on the same prompt and codes character portrayal on 8 category-pairs (coherent/complex/dynamic/closed vs their opposites); its headline finding is that LLM stories give characters tidier, more over-resolved arcs than human writing. The Freytag/Labov/Genette survey separately notes the field still lacks a unified way to score cross-disciplinary narrative form. Both are a plausible mechanism behind the already-documented phase-2 poetics transfer failure — weighted κ = 0.044 against the 0.60 bar, DRAMATIC-RECOGNITION-PLAN.md line 18 — and both were already independently flagged for exactly this cross-walk in workplan/inbox/2026-07-02-arxiv-2606.22454.md (still un-triaged).
Do: apply CASPER's closed/open-arc coding to the existing phase-2 calibration sample (config/poetics-calibration/phase2-*.yaml, already human-labeled) and correlate "closedness" of the learner-character's arc against the poetics rubric's cathartic_closure and anagnorisis dimension scores. If tidy/closed arcs correlate with high cathartic_closure but low anagnorisis, that's evidence the critic is rewarding resolution rather than genuine recognition — precisely the failure mode Dimension 3 (surprise_and_inevitability)'s anti-simulation guard was designed to catch, and a concrete target for strengthening it.
Evaluate: hand-coding the already-labeled phase-2 sample against CASPER's scheme is fully read-only. If instead an LLM classifier is used to code a larger, unlabeled transcript sample, that's a light paid classification pass (not a rescoring run) — flag separately for approval. existing data (read-only path available) LLM-coding path needs light API spend

P3 · Check the id-director register/charisma pilot against persona-geometry papers (2607.01034, 2607.02368)

Why: the Fluid Personality Framework argues conversational agents should jointly adapt both which persona is active and its expression intensity to the moment, rather than fixing either for a session. The id-director architecture (cells 101-109) already does something structurally adjacent — per-turn persona_delta and register selection persisted to id_construction_trace — but the cell_105/106 pilot (n=27 each, config/tutor-agents.yaml ~line 4654) found the verbose/charisma-tuned register beating the terse/pedagogy-tuned register on both rubrics, contradicting the expected intensity trade-off. The Dual Nature of Persona paper's finding — that persona expression under IPIP-50-style measurement is frame-dependent (its correlation geometry shifts under manipulated question ordering) rather than a fixed trait — is a direct methodological caution: if the charisma judge's persona_signature / affective_intensity scoring is itself sensitive to turn position or prior register history, the cell_105/106 null result could be partly a scoring-order artifact rather than a settled design finding.
Do: no new cell. Re-cross existing cell_101-109 rows: register-shift frequency (from id_construction_trace) against tutor_charisma_scores by turn index, checking for drift in persona_signature / affective_intensity that tracks turn position rather than actual register content — the within-instance geometry check 2607.02368 uses, applied to our own judge instead of a questionnaire.
Evaluate: existing rows only — id_construction_trace, tutor_charisma_scores, tutor_charisma_overall_score for cells 101-109, including the already-collected cell_105/106 pilot (n=27 each). A new analysis script, no new run. existing data
