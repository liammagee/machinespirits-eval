Prompt Engineering / Adaptive Tutoring

Learning to Prompt: Improving Student Engagement with Adaptive LLM-based High-School Tutoring UNBLOCK

Po-Chin Chang, Nicholas Hogan, Aske Plaat, Michiel T. van der Meer · arXiv 2606.20138 · June 18, 2026 · https://arxiv.org/abs/2606.20138

The paper introduces an adaptive prompt router trained on 14 pedagogical features (e.g., tutor scaffolding density, student understanding signals) extracted from conversation transcripts. A simulation benchmark shows the router outperforms two static baselines (p<0.001); an A/B deployment with 656 real conversations from 359 high-school students confirms sim-to-real transfer — the router learns to shift between analytical and scaffolding strategies, cutting interaction length by ~3 turns (p=0.007) while maintaining pedagogical quality.

Project relevance: The 14-feature transcript extractor is a ready-made operationalisation of dimensions the project tracks manually (scaffolding, Socratic questioning, affective mirroring). The sim-to-real approach maps directly onto the project's two-phase workflow: simulated learner runs (cells 1–79) → human pilot (§A1, IRB-pending). The router architecture — routing over fixed prompt templates rather than fine-tuning — is directly applicable to npm run prompt-lab -- autotune. Consider cross-referencing against the §6.3 suggestion-suite results: cells with higher adaptationIndex may implicitly be doing what this router does explicitly.

Agent Architecture / Knowledge Grounding

Navigating Unreliable Parametric and Contextual Knowledge: Explicit Knowledge Conflict Resolution for LLM Inference UNBLOCK

Huang Peng, Jiuyang Tang, Weixin Zeng, Hao Xu, Xiang Zhao · arXiv 2606.20245 · June 18, 2026 · https://arxiv.org/abs/2606.20245

Proposes MACR (Multi-Agent Conflict Resolution), a framework that rejects the standard binary paradigm of "trust the model" or "trust the context" and instead surfaces conflicts explicitly — between the LLM's parametric knowledge and external context, and among multiple external sources — then adjudicates them through a multi-agent reasoning step. Neither source is privileged a priori; the adjudicating agent weighs credibility, recency, and internal consistency before committing to a response.

Project relevance: The ego-superego architecture is exactly this pattern: the superego's critique is a conflict-resolution step between the ego's parametric priors and the in-context pedagogical constraints. MACR formalises what the project does implicitly and adds a third party (the adjudicator), which maps onto the judge model in the evaluation pipeline. The framework is a candidate design for cells 93–100 (superego variant ablations) — specifically for cell_98_two_pass and cell_99_coupling, where the conflict between ego and superego is currently handled by a single revision pass. An MACR-style explicit adjudication layer between ego and superego revision could be a cell_126+ addition.

Hierarchical Control in Multi-Agent Games: LLM-based Planning and RL Execution WATCH

arXiv 2606.20014 · June 18, 2026 · https://arxiv.org/abs/2606.20014

A two-layer hierarchical architecture: a pretrained LLM acts as a slow, centralized strategic controller that selects among specialised RL skill policies; the RL policies execute reactively at high frequency. Tested in a competitive 2v2 King of the Hill environment, the hybrid matches hand-crafted behaviour trees while significantly outperforming flat RL. The LLM layer operates on a longer timescale and sees the global game state; the RL layer handles local, real-time execution.

Project relevance: The timescale separation (slow strategic / fast reactive) resonates with the ego-superego deliberation loop: the superego operates at a "meta" timescale, critiquing holistic pedagogical intent; the ego executes the next utterance reactively. The cells 110–113 adaptive runner (LangGraph state-policy) already operationalise a version of this — this paper provides a formal grounding and a concrete benchmark for evaluating whether LLM strategic oversight improves over flat ego-only cells.
