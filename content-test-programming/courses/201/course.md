# Introduction to Programming

Welcome to Introduction to Programming. If this is your first time writing code, you're in the right place. We will not assume any background beyond reading English and knowing what a variable is in algebra (and we will reframe even that).

## Course Overview

What will we learn?

- **Variables and values** — how to give a name to a piece of data
- **Making decisions** — how to make a program do different things in different situations
- **Loops** — how to make the computer do repetitive work for you
- **Debugging** — what to do when your code doesn't work (which is most of the time)

## How We'll Learn

Programming is a skill, not a body of knowledge. You learn it by writing code, running it, getting it wrong, and figuring out why. Each lesson includes:

- A short explanation with small, runnable examples
- "Predict the output" exercises — read code before running it
- One or two practice programs to write yourself
- A debugging exercise — find the bug in a broken program

The most important habit to build: **when something doesn't work, slow down and read what's actually happening, rather than guessing a fix.** That habit is 90% of being a good programmer.
