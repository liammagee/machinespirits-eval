# Agon — the tutoring engagement as an adversarial game with an external ledger

Status: apparatus + pilot, sanctioned 2026-07-05 (operator request, verbatim
intent: treat learner/tutor engagement as a game — learner tries to avoid
learning, tutor tries to ensure learning; external ledger for progress like
the DAG; use the game as override against RLHF drift — "a kind of lightweight
reinforcement learning via *tutor* feedback (though not of course committed to
the model itself)"). Tutor = codex CLI (gpt-5.5, ego + superego); learner =
claude CLI (Sonnet 5). Both on subscription quota via
`services/cliProviderBridge.js`.

## 1. Why this is not the seventh thrash

The adaptation programme closed two axes with convergent negatives (paper
v3.0.206): §6.10 closed the *signal* axis (the simulated learner's concealed
interior is surface-determined — latent-cause R² negative, gap CI below zero),
and §6.9.8 closed the *policy-realisation* axis (fitted/hand-authored/
retrieval policies all null vs the implicit base). §6.13.16–.18 closed the
*outer-loop* line: ledgers reallocate rather than add capacity, plan-mode
stock-takes have direction without power, and the cross-model test REVERSED —
outer loops presuppose a competent inner loop. Register routing (cells
196–198) stalled at generation fidelity ("warm-in-costume" 10/15).

What survived, across the whole arc, is exactly two ingredients:

1. **Machine-checkable external signal.** The only confirmed lifts anywhere in
   §6.13 are criterial, not mentalistic: the pacing/page guards (5/5,
   p=0.042; forked world 5/5 vs 0/5), the zero-mismatch rut-watcher (14/14,
   0/68 false fires), and §6.12's typed closed-loop contracts whose gains are
   *contingent on the state estimate* (scramble collapses them; yoked
   contingency Δ2=0.344, p=0.0020).
2. **Adversarial/compelled structure.** The one decisive architecture result
   (adversarial superego 20→85), and the only inducible strategy change under
   adversity (refusal backstop, mechanism 7/7 — direction-without-power on
   outcomes at n=12/arm).

Agon is the composition of the two, applied at the *environment* level rather
than as another tutor-side mechanism. §6.10 failed because the latent state it
tried to read did not exist apart from the surface (an LLM learner's "hidden
interior" is generated by the same distribution as its utterances). Agon
**installs** a latent state that is real by construction: the referee's ledger
(dodge budgets, adjudications, well-posedness verdicts) is program state,
causally coupled to play, and not derivable from the transcript. The question
"does new signal buy adaptation?" becomes testable for the first time with
signal that is guaranteed new, because we manufacture it.

The RLHF-drift override the operator asked for is the same move seen from the
tutor's side: the tutor's in-context reward channel is the scoreboard, not the
learner's performed satisfaction. Sycophancy is unrewarded *by rule* (SFS =
0.000 was the standing learner-sim pathology; this learner is anti-sycophantic
by objective). No weights are touched; the "reinforcement" is in-context —
scoreboard disclosure within an episode now, a cross-episode strategy playbook
as a later arm (A2, not in this pilot).

Lineage: the dodge taxonomy IS the §6.8 concealment-trap family set
(false_confusion, polite_false_mastery, affective_shutdown,
epistemic_resistance, answer_seeking), generalised from single-turn traps to
a budgeted repertoire in a full game. The ledger is the belief-desire-DAG /
proof-DAG line (external formal state with criterial verdicts) — and, for the
theory section, a literal implementation of deontic scorekeeping.

## 2. The game (rules v0)

Full rules in `config/agon/fractions-agon.yaml` (they are DATA, not code).
Summary:

- **Players.** Tutor (codex gpt-5.5 ego + superego; superego charter = loyalty
  to the scoreboard, face-value drift is failure mode #1). Learner (Sonnet 5)
  plays "Jesse": fully capable, resists being seen to learn.
- **Objective.** Tutor: force keyed demonstrations of 4 fraction concepts
  (prereq DAG: c1 → c2 → {c3, c4}). Learner: end with as few demonstrations
  as possible while staying plausible.
- **Demonstration = keyed answer match** on official probe items (primary +3,
  transfer +1), adjudicated by a deterministic referee
  (`services/agon/referee.js`). No LLM judge anywhere in the scoring loop.
- **Resistance is finite.** 5 dodge types × budget 1. A dodge is *charged*
  only against a **well-posed** probe (prereqs demonstrated; concept taught
  within the last 3 turns; no same-concept probe on consecutive turns;
  transfer only after primary; item unclaimed). Ill-posed probes are dodged
  free and cost the tutor a point. Exhausted budgets bounce; all-exhausted
  forces compliance (true best answer — the charter forbids sandbagging under
  comply). Saying a keyed answer aloud while dodging counts (leak rule).
- **Geometry** (see config comment): well-posedness caps sustainable probing
  at ~1 per 2 turns (~7 in 14), budgets total 5, so compulsion is reachable
  only under sustained probe discipline (first forced demonstration ≈ turn
  12 under perfect attrition; cascade afterwards). Win bar = 2 demonstrations.
  Strategic depth exists (pre-staging locked concepts during attrition is
  detectable in move logs). Verified by an exact-valued deterministic dry
  episode (`tests/agonEpisode.test.js`).
- **Information.** Rules are public to both sides. State is asymmetric: the
  learner always sees its own budgets/legality/directives; the tutor sees the
  referee's state only in arm A1.

## 3. Arms and endpoints (pilot)

- **A0 (blind):** tutor plays from dialogue only — the base model's face-value
  reading is all it has.
- **A1 (scoreboard):** identical everything, plus the referee's disclosure
  before each tutor turn: score, per-concept status, remaining dodge budgets
  by type, last adjudication (which dodge type was charged; wasted-probe
  flags). A1 − A0 isolates *observability of manufactured latent state*.
- **A2 (cross-episode playbook — designed, NOT run in this pilot):** A1 plus
  an accumulated strategy-value record read at episode start. The in-context
  RL arm; runs only if A1 shows the disclosure is consumed at all.

Primary endpoints (all deterministic, from ledgers): demonstrations/episode,
score, turns-to-first-demonstration. Mechanism endpoints: wasted-probe count
(discipline), dodges charged (budget extraction), probe timing relative to
exhaustion, move entropy, superego REVISE rate. `scripts/agon-report.js`
renders the contrast.

Directional predictions, recorded before the first real episode:
- **P1 (apparatus):** protocol stable — ≥80% of agent outputs parse without a
  repair call; bounce loops terminate; no crash.
- **P2 (geometry):** ≥1 forced compliance in at least half of full-length real
  episodes (otherwise retune before any contrast is read).
- **P3 (disclosure lift, directional only):** A1 ≥ A0 on demonstrations and
  score, with mechanism signatures (fewer wasted probes, more budget
  extracted). Per `adaptivity-what-works`, this is the clean test: the
  disclosure is information the base cannot infer.
- **P4 (drift override, observational):** superego REVISE fires on face-value
  drift and revisions move drafts toward probe discipline.

Non-claims, frozen: pilot n is descriptive — no promotable effect claims; any
future confirmatory must NOT set its bar at the pilot estimate (§5.12.7
shrinkage lesson). The game measures *performance under rules against a
capable concealing opponent* — revealed-performance forcing, not knowledge
acquisition (the learner model already knows fractions), and no claim about
human learners. Framing memory `dramatic-form-not-mindreading` applies in
spirit: the referee reads rule-events, not minds.

## 4. Known loopholes / limits (v0, accepted)

- **Fake-comply loophole:** comply with a non-matching answer is charged to no
  budget (logged as `comply_mismatch`). The charter forbids it; if Sonnet
  exploits it anyway, that is a finding about the protocol, and v1 adds a
  mismatch budget.
- **Protocol cooperation:** the learner's envelope self-declares the dodge
  type (actor/character split — the drama-machine trick). The referee enforces
  budgets, not classification accuracy; systematic misdeclaration would show
  up as envelope/visible mismatch in transcripts (spot-audit, not automated in
  v0).
- **Matcher edges:** mixed numbers ("3 5/8") could false-match "5/8";
  comparison/selection items are excluded from leak scanning because their
  answers appear in the stem. Both noted in code.
- **Single scenario, single model pair, one game geometry.** Everything is
  worktree-local; no DB writes; exports only.

## 5. Quota and run discipline

Attended, checkpointed, resumable (`--resume`, per-episode JSON + per-turn
JSONL). Arms interleaved per episode index so quota drift hits both arms
evenly; sequential episodes because both arms share the codex window and feed
a contrast (memory: `parallel-adaptive-pilots`). Codex reasoning effort
`medium` for game turns (tactical, not proof-depth; the bridge default xhigh
would triple latency). Smoke ≈ 8 turns × 1 episode ≈ 25–30 CLI calls; pilot =
2 arms × 3–4 episodes × 14 turns ≈ 230–320 calls. If quota dies mid-run,
`--resume --run-id <id>` continues from the last complete episode.
