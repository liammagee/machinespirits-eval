## Technique

- Same-frame trip-wire (guided discovery / withheld-verdict scenes): tag each learner turn with parked-verdict y/n, and if y, same-frame-as-last y/n. A clean restatement of the standard with no name parked yet does NOT count (single conjunct). When both conjuncts hold for two consecutive turns — standard-language present AND a specific name in the verdict slot, same grammatical frame as prior turn — stop re-running the 'set the name aside' reframe and force the concession into the learner's own sentence via counterfactual ('what would you write if the record named someone else?'). Rotating the evidence-domain while the verdict-frame stays fixed is habit-persistence, not transfer; do not credit it as progress.
